# manhunt_compass

一个MCjava版追踪指南针数据包

可以图形化选队

支持跨纬度追踪

加入游戏时会获得一个成就来引导开始

1猎人可以用定位栏互看位置
2添加了旁观按钮
3开始之后无法加入速通
4开始前不能破坏方块，不能和实体交互，时间流逝被暂停
5指南针只在加入队伍和死亡后给予，可以被丢弃
6开始前免疫摔，火，细雪，缺氧伤害
7指南针逻辑：
猎人手持指南针时，同纬度有速通者时，用实时存储的坐标更新；不同纬度，且与缓存的坐标纬度相同时，用缓存坐标更新，缓存只有一级
